Set variable:

ENV_HOST_NAME - hostname
ENV_CENTOS_USER - centos user
ENV_PWD_CENTOS_USER - password for centos user
ENV_FTP_USER - ftp user 
ENV_PWD_FTP_USER - password for ftp-user 
ENV_WEBMIN_ADMIN - root user for webmin
ENV_PWD_WEBMIN_ADMIN - password for webmin root
ENV_WEBMIN_USER - user for webmin
ENV_PWD_WEBMIN_USER - password for webmin user
